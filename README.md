# Patients
Система хранения данных о пациенте с помощью БД MYSQL.

Система позволяет записывать, хранить и получать информацию о пациентах, содержащуюся в таблице базы данных, такую
как ID пациента, имя, фамилия, дата рождения, возраст и контактные данные. Также есть возможность добавлять данные о пациенте. Помимо этого, реализован пользовательский интерфейс в командной строке в виде меню. 

При запуске приложения пользователь попадает в главное меню и может выбрать следующие команды:
connect - Подключение к базе данных
exit - Выход из приложения

При выборе первого варианта пользователю предлагается создать базу данных и таблицу.
Затем пользователь попадает в подменю, в котором ему доступно взаимодействие с базой данных. Пользователь может выбрать следующие команды:
Add data from "txt" file - Позволяет добавить данные из текстового файла
Add data from console - Позволяет добавить данные из консоли
Show data from the database - Выводит на экран данные
Disconnect from the database - Отключает пользователя от базы данных
Delete the table - Удаляет таблицу
Delete the database - Удаляет базу данных
Back - Позволяет вернуться в меню
