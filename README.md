# Patients
Система хранения данных о пациенте с помощью БД MYSQL.

Система позволяет записывать, хранить и получать информацию о пациентах, содержащуюся в таблице базы данных, такую
как ID пациента, имя, фамилия, дата рождения, возраст и контактные данные. Также возможно добавлять данные о пациенте 
и удалять их. Помимо этого, реализован пользовательский интерфейс в командной строке в виде меню.
