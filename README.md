# Patients
Система хранения данных о пациенте с помощью БД MYSQL.

Система позволяет записывать, хранить и получать информацию о пациентах, содержащуюся в таблице базы данных, такую
как ID пациента, имя, фамилия, дата рождения, возраст и контактные данные. Также есть возможность добавлять данные о пациенте. Помимо этого, реализован пользовательский интерфейс в командной строке в виде меню. 

При запуске приложения пользователь попадает в главное меню и может выбрать следующие команды:
connect - Подключение к базе данных
exit - Выход из приложения

При выборе первого варианта пользователю предлагается создать базу данных и таблицу.
Затем пользователь попадает в подменю, в котором ему доступно взаимодействие с базой данных. Пользователь может выбрать следующие команды:

Add data from "txt" file - Позволяет добавить данные из текстового файла

Add data from console - Позволяет добавить данные из консоли

Show data from the database - Выводит на экран данные

Disconnect from the database - Отключает пользователя от базы данных

Delete the table - Удаляет таблицу

Delete the database - Удаляет базу данных

Back - Позволяет вернуться в меню

При авторизации пользователь вводит следующие данные: хост, имя пользователя, пароль, порт и клиентский флаг. 
Также по необходимости следует ввести имя базы данных и имя таблицы.

Для работы меню используются следующие функции:

DrawMenu, DrawEdit, DrawArrow - функции, отрисовывающие меню

Для хранения данных о пациенте используется класс DataExtraction. Этот класс получает текстовую строку, проверяет её корректность
и преобразует элементы строки в данные таблицы. Также класс содержит конструктор по умолчанию и методы, возвращающие
данные таблицы. 

В случае возникновения исключений любого типа будет вызываться исключение простейшего класса DataExtractionException.

Для работы с базой данных и таблицами были написаны следующие функции:

1.void pull_data(std::list<DataExtraction>& a, const std::string& input)

Функция считывает данные из файла и добавляет в таблицу

\param a - Список данных о пациентах

\param input - Имя файла

2.void show_data_from_cont(std::list<DataExtraction>& a)

Функция выводит данные о пациентах в консоль

\param a - Список данных о пациентах

3.void create_db(MYSQL* mysql, const std::string& db)

Функция генерирует новую базу данных

\param mysql - Дескриптор БД

\param db - Имя БД

4.void delete_db(MYSQL* mysql, const std::string& db)

Функция удаляет базу данных

\param mysql - Дескриптор БД

\param db - Имя БД

MYSQL* connect_to_DB(const std::string& host1, const std::string& user1,
    const std::string& passwd1, const std::string& db1, const unsigned int port1, const unsigned long clientflag1)

Функция осуществляет подключение к базе данных

\param host1 - Хост

\param user1 - Имя пользователя

\param passwd1 - Пароль

\param db1 - Имя БД

\param port1 - Порт

\param clientflag1 - Клиентский флаг
